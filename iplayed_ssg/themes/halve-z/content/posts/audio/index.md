## Usage

- `src`: an array of audio file paths (mandatory)

### Example
#### Input

```rs
{{/* audio(src=["over9000.ogg", "over9000.mp3"]) */}}
```

#### Output

```html
{{ audio(src=["over9000.ogg", "over9000.mp3"]) }}
```

{{ audio(src=["over9000.ogg", "over9000.mp3"]) }}
